Welcome to my Angular application!

This is a personal project that I'm undertaking to gain additional experience with developing applications, user interfaces, and user experiences with the Angular framework/platform. Current, I have the following features implemented (and plan to add more as time allows):

  • Utilizes Angular Material UI components to take advantage of the Material design system (created by Google).
  • Multiple Angular Material themes (currently light and dark) that follow the guidance of the Material design spec.
  • Visual styles defined with Sass (preprocessor scripting language that compiles to CSS).
  • Unit testing with the Karma test runner and the Jasmine test framework.
  • Continuous deployment with Travis CI and Heroku.
  • View (a.k.a. page) transition animations with the Angular animations system.
  • Responsive layouts (please try resizing your browser).
  • Content (e.g. my work history) populated with JSON files.

The next features that I would like to implement are passing the current theme to routed views, fetching/processing data from a 3rd party API, and allowing users to create accounts and log in.
